About This Project

Crop Recommendation System

This system uses machine learning to recommend the best crop based on:

  • Nitrogen (N) content in soil
  • Phosphorous (P) content in soil
  • Potassium (K) content in soil
  • Temperature
  • Humidity
  • Soil pH
  • Rainfall
The model was trained using Random Forest algorithm on a comprehensive dataset.
